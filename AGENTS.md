# Contributing notes

`@ancplua/typespec-maf` — TypeSpec emitters for Microsoft Agent Framework's
declarative agent and workflow YAML, verified by loading the output through
Microsoft's own agent and workflow loaders. See [`docs/PRD.md`](docs/PRD.md) for
the design and roadmap.

## Layout

- `lib/` — the `@ancplua/typespec-maf` library: `src/index.ts` (decorators, the
  emitter `$onEmit`, `$onValidate`, the PowerFx lint rule), `lib/main.tsp`
  (decorator declarations).
- `main.tsp` — sample agents and workflows.
- `tspconfig.yaml` — emit + linter config.
- `generated/maf-actions.gen.tsp` — generated D2 action shapes (tracked).
- `tools/maf-model-gen/` — the action-shape generator.
- `tests/MafRoundTrip/`, `tests/MafWorkflowRoundTrip/` — .NET round-trip harnesses.
- `versions.props` / `Directory.Build.props` — .NET package pins + build conventions.

## Build & test

```bash
npm install
npm run compile          # emit agent + workflow YAML from main.tsp
npm run lint             # compile with --warn-as-error
npm run test:roundtrip   # D1 agent round-trip (Microsoft's CreateFromYamlAsync)
npm run test:workflows   # D2 workflow equivalence + DeclarativeWorkflowBuilder build
npm run test:powerfx     # PowerFx reference matrix
npm run test:actions     # generated action lib compiles + spot-check
```

## Generated code

`generated/maf-actions.gen.tsp` is produced by `tools/maf-model-gen` reflecting over
the pinned `Microsoft.Agents.ObjectModel` (version in `versions.props`). It is
committed but `// <auto-generated/>` — don't hand-edit. To absorb MAF changes: bump
`ObjectModelVersion`, run `npm run gen:actions`, run `npm run test:actions`, and
review the diff.

## Conventions

- **Round-trip or it doesn't ship:** an emitter change is mergeable only with the
  Microsoft-loader round-trip test green against the pinned MAF version.
- **Pinned versions:** MAF packages and the TypeSpec toolchain are pinned; no
  floating `latest`.
- Verify MAF API shapes against the actual Microsoft Agent Framework source rather
  than the docs, which lag the source.
